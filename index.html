Internationalize your applications Laravel

Laravel4-lang is a repository on GitHub and Composer which contains files of languages of the PHP framework, Laravel. The official project of Laravel just offers files of translation in english, Laravel4-lang aims to do others languages.

Read documentation to know how to install, configure and contribute.
